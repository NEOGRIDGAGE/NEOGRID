# NDAP Cloud Products

## 1. NDAP API
Developer access layer for core protocol operations. For teams integrating asset state and verification into applications.

## 2. NDAP Compute
Rust execution engine usage for cryptographic and state-processing workloads. For performance-sensitive workloads.

## 3. NDAP Storage
IPFS plus state history storage for verifiable asset records. For auditability and durable provenance.

## 4. NDAP Consensus
Enterprise validation layer for private and regulated deployments. For organizations needing deterministic finality and validator controls.

NDAP Cloud packages protocol capability into clear infrastructure products for different users and deployment needs.
