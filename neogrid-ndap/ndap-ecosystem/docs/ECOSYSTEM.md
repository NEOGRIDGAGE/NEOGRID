# NDAP Ecosystem

NDAP is a protocol, not a codebase.

- The spec defines truth.
- The reference implementation defines the behavior baseline.
- Clients are interchangeable implementations.
- Conformance tests define correctness.
- The simulator is a research tool only.

## Ecosystem Diagram
spec -> reference implementation -> clients -> conformance tests

The architecture is language-agnostic and allows multiple independent implementations.
