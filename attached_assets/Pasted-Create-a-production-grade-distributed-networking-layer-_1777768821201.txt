# 7. SYNC PROTOCOL (src/network/sync.js)

On peer connect or startup:

1. Send SYNC_REQUEST:
   {
     type: "SYNC_REQUEST",
     nodeId,
     lastKnownRoot
   }

2. Peer responds with SYNC_RESPONSE:
   {
     stateRoot,
     mmrRoot,
     recentLogs[]
   }

3. Local node:
   - validate logs
   - recompute SMT root
   - recompute MMR root
   - compare state

If mismatch:
   - request missing logs
   - resolve by highest valid proof chain
